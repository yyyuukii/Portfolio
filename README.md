# Yuki Xing – Project Portfolio

This repository highlights several computer science projects completed in UC Berkeley coursework. 
The projects demonstrate experience in **data structures, artificial intelligence, and simulation-based system design**.

Due to course policies at UC Berkeley, the full source code for some projects cannot be made publicly available. However, the design, algorithms, and implementation details are documented in each project description.

## Projects

### 👻 Pacman AI Agents
**UC Berkeley CS188 – Artificial Intelligence**

Implemented multiple AI agents to control Pacman in a grid-based environment.  
The project explores different decision-making strategies including classical search, adversarial search, and reinforcement learning.

Key topics:
- Search algorithms (DFS, BFS, UCS, A*)
- Adversarial search (Minimax, Alpha-Beta pruning, Expectimax)
- Reinforcement learning (Q-learning)

This project demonstrates the design and evaluation of intelligent agents in a dynamic environment.

---

### 🐜 Ants vs SomeBees Strategy Simulation
**UC Berkeley CS88C – Object-Oriented Programming**

Developed a tower-defense style simulation where ants defend their colony against invading bees.

Key concepts:
- Object-oriented system design
- Class hierarchies and inheritance
- Turn-based simulation mechanics

Different ant types implement unique abilities, requiring strategic placement and resource management to defeat the bees.

---

### 🗺️ Map Data Structure Implementation
**UC Berkeley CS88C – Data Structures**

Implemented a custom Map (key–value dictionary) data structure from scratch in Java.

The project compares multiple implementations and analyzes their performance:

- Unordered Linked List Map (ULLMap)
- Binary Search Tree Map (BSTMap)

Performance experiments demonstrate how different underlying data structures impact runtime complexity.
